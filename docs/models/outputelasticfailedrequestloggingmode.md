# OutputElasticFailedRequestLoggingMode

Data to log when a request fails. All headers are redacted by default, unless listed as safe headers below.


## Values

| Name                  | Value                 |
| --------------------- | --------------------- |
| `PAYLOAD`             | payload               |
| `PAYLOAD_AND_HEADERS` | payloadAndHeaders     |
| `NONE`                | none                  |
