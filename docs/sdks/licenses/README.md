# Licenses
(*licenses*)

## Overview

Actions related to licenses. The <code>/licenses</code> endpoints do not apply to Cribl.Cloud deployments.

### Available Operations

* [list_license](#list_license) - Get a list of License objects
* [create_license](#create_license) - Add a license to your deployment
* [get_license_by_id](#get_license_by_id) - Get License by ID
* [delete_license_by_id](#delete_license_by_id) - Delete License
* [get_license](#get_license) - Get license usage metrics, aggregated by day, up to last 90 days
