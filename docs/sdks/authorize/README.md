## get_authorize_policy

get the client's authorization policy

### Example Usage

```python
from cribl import Cribl
import os


with Cribl(
    server_url="https://api.example.com",
    bearer_auth=os.getenv("CRIBL_BEARER_AUTH", ""),
) as c_client:

    res = c_client.authorize.get_authorize_policy()

    # Handle response
    print(res)

```
